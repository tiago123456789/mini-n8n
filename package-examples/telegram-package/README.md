## TEMPLATE PACKAGE

This is a template package to use to create a custom node for mini-n8n.

## Installation

- Clone this repository
- Install dependencies, use the command:
```bash
pnpm install
```
- Build the package, use the command:
```bash
pnpm run build
```
- Publish the package on npm registry, use the command:
```bash
pnpm publish --public
```
